# BGD

Really mini 30min project to help you download cute backgrounds for your machine without quitting your fancy console!

## Installation 
```
$ git clone https://github.com/ghadd/bg-downloader
$ cd bg-downloader
$ sudo chmod +x bgd.sh
$ ./bgd.sh
```

## Testing out
```
$ bgd --help
```
